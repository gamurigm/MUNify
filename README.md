# MUNify v2 - AI Document Generation 🚀

MUNify es una plataforma avanzada diseñada para simulaciones de Modelos de Naciones Unidas (MUN), potenciada por agentes de Inteligencia Artificial y edición colaborativa en tiempo real.

![MUNify Preview](./assets/images/MUNify-main-v2.png)

## ✨ Características Principales

- **🤖 AI Agents**: Agentes diplomáticos autónomos (investigación, redacción, crítica y validación).
- **📝 Editor Colaborativo**: Edición en tiempo real usando TipTap y Yjs.
- **📄 LaTeX Export**: Generación de documentos oficiales (Position Papers, Resoluciones) con formato académico profesional.
- **🌍 Comités Multidisciplinarios**: Soporte especializado para más de 14 comités de la ONU.
- **⚖️ Lenguaje Diplomático**: Motor de IA entrenado para redacción formal exclusivamente en español.

## 🛠️ Stack Tecnológico

- **Frontend**: Next.js 15+, Tailwind CSS, Lucide React.
- **Backend**: Spring Boot (Java), PostgreSQL, Redis.
- **AI Agent Engine**: Python (FastAPI), LangGraph, NIM, Tavily Search.
- **Colaboración**: WebSocket, Yjs, TipTap.

## 🚀 Inicio Rápido

Para iniciar todos los servicios localmente:

```bash
.\start-munify.bat
```
